Safely evaluate expression with some supported functions and local variables.

Adapted from Python’s built in ast.literal_eval function, this function provides more numerical flexibility while trying to be as safe as possible.

  • locals (dict) – Only generic number types, numpy number types, and numpy.ndarray are supported as local variables.
